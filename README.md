### Installation
- Scripti indirmeniz sonrasında istediğiniz bir klasör içine koyabilirsiniz.
- server.cfg/resouces.cfg dosyanızın içine girip ensure ra1derBanking olarak startlayın.
- Dosya içinde ki sql dosyasını açıp sql verisini çalıştırın.
